# تعلم الحروف العربية 📚

تطبيق تفاعلي وممتع لتعليم الأطفال الحروف العربية من خلال الألعاب والأنشطة التعليمية.

## المميزات ✨

- **لعبة الاختيار**: اختر الحرف الصحيح من بين عدة خيارات
- **تعلم الحروف**: تعرف على كل حرف مع كلمة وصورة وصوت
- **حركات الحروف**: تعلم حركات الحروف (الفتحة، الضمة، الكسرة)
- **رسم الحروف**: مارس رسم الحروف على الشاشة
- **موسيقى تعليمية**: أصوات تشجيعية وموسيقى خلفية
- **واجهة ودية**: تصميم ملون وسهل الاستخدام للأطفال
- **يعمل بدون إنترنت**: تطبيق PWA يعمل حتى بدون اتصال بالإنترنت

## التثبيت 📱

### على الهواتف والأجهزة اللوحية:

#### iOS (iPhone/iPad):
1. افتح التطبيق في متصفح Safari
2. اضغط على زر المشاركة (Share)
3. اختر "Add to Home Screen"
4. أدخل الاسم واضغط "Add"

#### Android:
1. افتح التطبيق في متصفح Chrome
2. اضغط على القائمة (⋮)
3. اختر "Install app" أو "Add to Home Screen"
4. اتبع التعليمات

### على الحاسوب:

#### Windows/Mac/Linux:
1. افتح التطبيق في متصفح Chrome أو Edge
2. اضغط على أيقونة التثبيت في شريط العنوان
3. اتبع التعليمات

## الاستخدام 🎮

1. **اختر وضع التعلم**:
   - 🎮 لعبة الاختيار: لعبة تفاعلية
   - 📖 تعلم الحروف: دراسة الحروف بهدوء
   - ✨ حركات الحروف: تعلم الحركات

2. **استمع واتعلم**: استخدم أزرار الاستماع لسماع النطق الصحيح

3. **تفاعل**: اختر الإجابات الصحيحة أو مارس الرسم

## المتطلبات 💻

- متصفح حديث يدعم PWA (Chrome, Firefox, Safari, Edge)
- اتصال بالإنترنت للتحميل الأول (اختياري بعد ذلك)

## الملفات 📁

- `index.html` - الصفحة الرئيسية
- `style.css` - أنماط التصميم
- `game.js` - منطق اللعبة
- `audio.js` - إدارة الأصوات
- `data.js` - بيانات الحروف والكلمات
- `sw.js` - خدمة العامل (Service Worker)
- `manifest.json` - بيان التطبيق
- `icon.png` - أيقونة التطبيق
